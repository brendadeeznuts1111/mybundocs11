# My Bun Template

A simple starter template for Bun projects.

## Getting Started

1. Install dependencies:

   ```bash
   bun install
   ```

2. Run in development mode:

   ```bash
   bun dev
   ```

3. Run in production mode:

   ```bash
   bun start
   ```

4. Run with custom port:

   ```bash
   PORT=8080 bun dev
   ```

## What's included?

- Basic TypeScript setup
- Example HTTP server with configurable port
- Development scripts
- Ready to customize!

## Environment Variables

- `PORT`: Server port (defaults to 3000)
