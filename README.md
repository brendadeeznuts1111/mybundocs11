# My Bun Docs 11

A collection of Bun-related documentation, test projects, and examples.

## Repository Contents

This repository includes various Bun (JavaScript runtime) related materials:

- **`bun-docs-repo/`**: The main Bun documentation and source repository, including guides, API docs, benchmarks, and build scripts.
- **`comprehensive-api-test/`**: A comprehensive API testing project demonstrating Bun's server capabilities.
- **`enhanced-api-test/`**: Enhanced API testing with file upload functionality.
- **`test-project/`**: Collection of test projects showcasing different Bun features like HTTP servers, fetch API, and bundling.
- **`your-project/`**: A basic Bun project template.
- **`final-test-project/`**: Final test implementation.
- **`full-api-project/`**: Full-featured API project.
- **`test-port-project/`**: Port testing utilities.
- **`enhanced-project/`**: Enhanced project setup.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) runtime installed

### Running Projects

Each subdirectory contains its own `package.json` and can be run independently:

```bash
cd <project-directory>
bun install
bun run dev  # or bun run test, etc.
```
