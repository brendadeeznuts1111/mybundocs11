# My Bun Template

A simple starter template for Bun projects.

## Getting Started

1. Install dependencies:

   ```bash
   bun install
   ```

2. Run in development mode:

   ```bash
   bun dev
   ```

3. Run in production mode:

   ```bash
   bun start
   ```

## What's included?

- Basic TypeScript setup
- Example HTTP server
- Development scripts
- Ready to customize!
