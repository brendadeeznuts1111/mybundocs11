# Contributing

## This is a Personal Repository

This repository contains personal documentation and examples for learning Bun. It is not actively seeking external contributions at this time.

## How to Contribute

While this repository is primarily for personal use, you're welcome to:

- **Fork the repository** and adapt the examples for your own projects
- **Report issues** if you find bugs in the example code
- **Suggest improvements** through GitHub issues

## Code of Conduct

Please be respectful and constructive in any interactions related to this repository.
